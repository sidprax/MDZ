# MDZ Antigravity Plugin

This plugin bundles MDZ Token Advisor for Antigravity:

- skill instructions,
- MCP server config,
- lifecycle hooks,
- behavior rules.

Install with:

```bash
npm run mdz -- install antigravity --scope project
```

or:

```bash
npm run mdz -- install antigravity --scope user
```

Then refresh Antigravity customizations and MCP servers.
